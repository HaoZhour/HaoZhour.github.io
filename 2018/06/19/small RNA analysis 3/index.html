mirdeep2 的安装

Posted on 2018-06-19 | In Small RNA-seq data analysis

mirdeep2为最常用的miRNA预测软件，安装要先把下面4个包和软件装好

  • a bowtie short read aligner
  • b Vienna package with RNAfold
  • c.i SQUID library
  • c.ii randfold
  • d Perl package PDF::API2

在无权限的服务器里前前后后花费了半个月没装下来，软件依赖其它包，包依赖其它包，太难了！

期间尝试过一下做法：

  1. 学习了如何安装perl 模块，但是对于PDF::API2这个模块，不知为啥不能用

perl模块安装大全

  1. 想过曲线救国，用另一预测软件mirdentify，先把reads 比对到其它物种的mature RNA，在用剩下reads预测，这样也能达到类似miredeep的效果，但是 可能这软件筛选条件太严格，预测出0个。
  • 下这个软件的时候也遇到两个事，源代码在主页上下不下来了，写邮件给他，友好的处理好了。
  • 这个软件依赖，是输出PDF格式需要的cairo-lib.也是一直在服务器上没下下来。编译的时候用–always-make 选项忽略了这个包，也算是编译成功了，但PDF文件输出的这个功能没了。
  1. 用mac，在有root的权限的情况下，还是没能装下来。
  • Vienna package安装老报错。编译器的问题。

    在谷歌上找原因，忽然搜出了 bioconda install mirdeep2

功夫不负有心人，想到了之前关于bioconda的推文，马上下了bioconda，几分钟就把mirdeep2下载下来了~。
